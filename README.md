## About Hivet!
<p align="justify"> Aplikasi berbasis Android yang berperan sebagai wadah dalam mempertemukan dokter hewan dengan para pemilik hewan peliharaan untuk dapat saling berkomunikasi dan dapat melakukan konsultasi kesehatan hewan secara online. Selain fitur konsultasi online, HiVet! juga menawarkan fitur lain berupa Janji Temu dengan dokter hewan.</p>

## Tech Stack
<ul>
<li> Kotlin</li>
<li> Firebase </li>
<li> Payment Gateway (Midtrans) </li>
</ul>
